# Wellness Log Frog 🐸🌿

## About the Project
**Wellness Log Frog** is a wellness tracking app designed to help users monitor their daily water intake and track wellness behaviors. Built with **MongoDB**, this application securely stores user inputs, providing an interactive and engaging experience in a **lush green forest-themed** UI.

## Features
- 🔐 **Secure Login System** – Users can sign up and log in to manage their personal wellness data.
- 🐸 **Water Tracking** – Log daily water intake in cups and monitor hydration levels.
- 📊 **Wellness Data Storage** – All user inputs are stored in **MongoDB** for long-term tracking.
- 🌱 **Forest-Themed UI** – Dark, lush green forest aesthetic with animations.
- 💃 **Dancing Frog Animation** – A fun frog appears when users successfully log their data.
- 🖱️ **Flower Mouse Cursor** – A unique interactive mouse that appears as a flower.

## Tech Stack
- **Frontend:** React, React Router, CSS (dark forest theme)
- **Backend:** Node.js (to be implemented)
- **Database:** MongoDB (to store user wellness data)
- **Authentication:** To be integrated (e.g., Firebase or JWT-based authentication)

## Installation & Setup
### Prerequisites
- Install **Node.js** and **npm** if not already installed.
- MongoDB setup (if using locally).

### Steps to Run Locally
```sh
# Clone the repository
git clone https://github.com/your-username/wellness-log-frog.git
cd wellness-log-frog

# Install dependencies
npm install

# Start the application
npm start
```
